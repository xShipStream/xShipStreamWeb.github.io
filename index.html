Installation

Einrichten in Kodi

Über dieses Repository können xStream und xShip ganz normal installiert werden. Du installierst zuerst das Repository-ZIP und anschließend die Add-ons daraus.

1
Repository ZIP installieren
Repository ZIP herunterladen und in Kodi über „Aus ZIP-Datei installieren“ hinzufügen.
2
Repository öffnen
In Kodi unter „Aus Repository installieren“ das xShipStream Repo auswählen.
3
xStream & xShip installieren
Die Add-ons wie gewohnt aus dem Repository installieren und aktualisieren.
